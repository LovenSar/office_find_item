Office Find Item - Win7 32-bit 文件内容搜索工具

用法:
  ofind.exe -ui
  ofind.exe -roots "D:\Docs;E:\Work" -q "合同编号：A-001" [-workers 8] [-open 1]

参数:
  -daemon
    	内部使用：常驻索引+缓存进程（stdin 控制，stdout JSON Lines）
  -open int
    	搜索结束后打开第N个结果（从1开始），0表示不打开
  -q string
    	Query 1：要查找的字符串（Unicode）
  -q2 string
    	Query 2：要查找的字符串（交集）
  -q3 string
    	Query 3：要查找的字符串（交集）
  -roots string
    	要搜索的根目录，多个用 ; 分隔
  -ui
    	启动Windows UI
  -worker
    	内部使用：作为子进程执行搜索并输出 JSON Lines
  -workers int
    	并发工作线程数（默认=CPU核心数）

说明:
  - 默认支持 txt/md 等文本、docx/xlsx/pptx；doc/xls/ppt/pdf 通过系统 IFilter（需已安装对应组件）
  - 结果可用 -open N 在资源管理器中选中
